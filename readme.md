# Pranav Verma - Portfolio Website

A modern, responsive portfolio website built with React, Vite, and Tailwind CSS.

## Features

- Modern, clean UI design similar to shubhporwal.me
- Smooth animations using Framer Motion
- Responsive design for all devices
- Sections: Hero, About, Experience, Projects, Skills, Education, and Contact

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Add your project screenshots to the `public` folder:
   - `ecommerce.png` - E-Commerce Website screenshot
   - `blog.png` - Full-Stack Blog Application screenshot
   - `gemini.png` - Gemini Clone screenshot
   - `movie.png` - Movie Browser App screenshot

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
