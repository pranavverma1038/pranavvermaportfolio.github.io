# Project Screenshots

Please add your project screenshots to this folder with the following names:

1. `ecommerce.png` - E-Commerce Website screenshot
2. `blog.png` - Full-Stack Blog Application screenshot  
3. `gemini.png` - Gemini Clone screenshot
4. `movie.png` - Movie Browser App screenshot

The images will be automatically displayed in the Projects section. If an image is missing, a placeholder will be shown instead.
